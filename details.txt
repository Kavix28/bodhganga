4. Security System

JWT Access Tokens (15 min)

JWT Refresh Tokens (30 days)

BCrypt password encryption

Google OAuth2 login

Role Based Access Control

CSRF protection

Secure cookies (HttpOnly + Secure)

5. Preview / Paywall Logic

Answers store preview and full text separately

Preview is always public

Full text returned only if:

user has transaction success OR

active subscription

6. Payment Flow

User clicks Buy

Backend creates Transaction

Calls Stripe/Razorpay session

Provider redirects user

Webhook confirms payment

Access is granted
